Experimental Methodology

DISCLAIMER: This project is an experimental analysis tool. The methodology described below is a best-effort attempt to categorize on-chain data. It has NOT been wargamed against adversarial edge cases. Use at your own risk.

Data Source: We run a direct node script against the Aztec Continuous Clearing Auction (CCA) contract on Ethereum Mainnet. We scan the last 200,000 blocks (approx. 30 days) to capture both the public window and early contributor deposits.

"Market" vs "Limit" Classification:

  • Market Bids (The Floor): Bids with a limit price > 10,000,000 ETH FDV. These are effectively "Infinity" bids, indicating price insensitivity.
  • Limit Bids: Bids with a specific price sensitivity below the "Market" threshold.
